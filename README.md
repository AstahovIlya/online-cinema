## О проекте

![Product Name Screen Shot](https://github.com/AstahovIlya/online-cinema/blob/main/Screenshot.png)

Цель учебного проекта - реализовать базовый функционал кинотеатра и научиться применять сторонние API.

Что было реализовано:
* Базовая фильтрация на полнометражные фильмы и сериалы
* Бесконечная лента на главной странице
* Переход на страницу конкретго фильма с использованием локального хранилища
* Динамическое создание и заполнение сведений в HTML

Дальнейшие планы на проект:
* Реализация глубоких фильтров
* Создание личного кабинета
* Создание подборок
